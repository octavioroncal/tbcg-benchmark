protected class AnnotationItemProvider.ReferencesPropertyDescriptor extends ItemPropertyDescriptor
A specialized property descriptor for the annotation references feature. Specialized annotation item providers created by an annotation item provider adapter factory can override the value choices to provide more restricted choices. But they won't generally need to do that because this implementation delegates to the assistant.
