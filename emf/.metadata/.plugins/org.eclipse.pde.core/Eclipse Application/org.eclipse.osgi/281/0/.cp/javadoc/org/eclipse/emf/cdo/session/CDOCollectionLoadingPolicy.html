Package org.eclipse.emf.cdo.session

Interface CDOCollectionLoadingPolicy

All Superinterfaces:
CDOListResolver
public interface CDOCollectionLoadingPolicy extends CDOListResolver
A strategy that specifies which list elements must be present (loaded) in a CDOID list of a revision when a certain list index is accessed. Implementations of this interface can control the exact characteristics of a certain partial collection loading strategy.
